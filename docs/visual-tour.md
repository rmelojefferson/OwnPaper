# Tour visual

Esta seção será atualizada com capturas revisadas do site público e do painel administrativo.

## Onde colocar os prints

Salve os arquivos em `docs/assets/screenshots/` com estes nomes:

| Arquivo | Tela recomendada |
| --- | --- |
| `site-home.png` | Home do site público |
| `site-publicacoes.png` | Listagem pública de publicações |
| `site-publicacao.png` | Página de uma publicação |
| `site-autores.png` | Página pública de autores |
| `admin-home.png` | Tela inicial do painel admin |
| `admin-publicacoes.png` | Página administrativa de publicações |
| `admin-estatisticas.png` | Página de estatísticas |
| `admin-perguntas-quiz.png` | Página de perguntas do quiz |
| `admin-configuracoes-site.png` | Hub de configurações do site |

## Critérios para os prints

- Use PNG.
- Prefira largura entre 1440 e 1920 px.
- Evite dados reais sensíveis: e-mails, nomes internos, mensagens privadas, tokens, logs e configurações SMTP.
- Capture telas já revisadas visualmente.
- Se possível, use conteúdo demonstrativo consistente, sem imagens quebradas.

Depois que os arquivos estiverem no diretório, a galeria pode ser reativada nesta página.
