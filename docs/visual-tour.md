# Tour visual

Esta seção será atualizada com capturas revisadas do site público e do painel administrativo.

Enquanto os prints finais não entram na documentação, esta página funciona como checklist do material visual.

## Onde colocar os prints

Salve os arquivos em `docs/assets/screenshots/` com estes nomes:

| Arquivo | Tela recomendada |
| --- | --- |
| `site-home.png` | Home do site público |
| `site-publicacao.png` | Página de uma publicação |
| `site-doacoes.png` | Página pública de apoio/doações |
| `site-cookies.png` | Página pública de cookies |
| `admin-home.png` | Tela inicial do painel admin |
| `publicacoes.png` | Página administrativa de publicações |
| `editor-publicacao.png` | Editor de publicação |
| `fluxo-editorial.png` | Fluxo editorial de uma publicação |
| `midias-pendentes.png` | Mídias pendentes/quarentena |
| `perguntas-quiz.png` | Perguntas reutilizáveis do quiz |
| `contato-inbox.png` | Inbox de contato |
| `newsletter.png` | Newsletter, templates ou disparos |
| `estatisticas.png` | Página de estatísticas |
| `configuracoes-site.png` | Configurações do site |

## Critérios para os prints

- Use PNG.
- Prefira largura entre 1440 e 1920 px.
- Evite dados reais sensíveis: e-mails, nomes internos, mensagens privadas, tokens, logs e configurações SMTP.
- Capture telas já revisadas visualmente.
- Se possível, use conteúdo demonstrativo consistente, sem imagens quebradas.
- Não inclua dados reais sensíveis, como credenciais SMTP, tokens, chaves, backups, e-mails privados ou logs internos.

Depois que os arquivos estiverem no diretório, a galeria pode ser reativada nesta página.
