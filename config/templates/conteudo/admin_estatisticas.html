Como estas estatísticas funcionam

Estatísticas internas: {% if estatisticas_config.internas_ativas %}ativas{% else %}desativadas{% endif %}. Elas só são registradas quando o visitante aceita cookies opcionais.

  • Retenção dos agregados diários{{ estatisticas_config.retencao_agregados }} dias
  • Retenção dos eventos brutos{{ estatisticas_config.retencao_brutos }} dias

Eventos brutos são sinais individuais de permanência, como caminho acessado, início da sessão, último sinal e duração aproximada. Eles usam hash de sessão e não registram nome, e-mail, usuário do painel ou IP nessa estatística.

Para analytics profundo, use ferramentas dedicadas como Plausible, Umami, Matomo, Cloudflare Analytics, Google Analytics ou Tag Manager. O OwnPaper evita campo de script livre; as integrações são por campos estruturados para reduzir risco de script malicioso.
